Install Windows
with USB & Rufus

The ultimate, visually stunning guide to creating a bootable USB and installing Windows 11 (or any version) with ease.

Get Started

What You’ll Need

USB Drive

At least 8GB, empty or backed up.

Windows ISO

Download from Microsoft.

Rufus

Get the latest version from rufus.ie.

Step-by-Step Guide

  1. 1

    Download the Windows ISO

    Go to the Microsoft Windows Download page. Select your Windows version and click "Download". Save the ISO file to your computer.

    Tip: For Windows 10 or 8, use their respective download pages.

  2. 2

    Download and Open Rufus

    Visit rufus.ie and download the latest version of Rufus. It’s a portable app—just double-click to open, no installation required.

  3. 3

    Insert Your USB Drive

    Plug your USB drive into your computer. Warning: All data on the USB will be erased, so back up anything important first!

  4. 4

    Configure Rufus

    • Device: Select your USB drive from the dropdown.
    • Boot selection: Click "SELECT" and choose your Windows ISO file.
    • Partition scheme: Choose GPT for modern UEFI systems, or MBR for older BIOS systems.
    • File system: Use NTFS (for files over 4GB) or FAT32 (for maximum compatibility).
    • Volume label: You can name your USB (e.g., "WIN11USB").

    Advanced: Leave other settings at default unless you know what you’re doing.

  5. 5

    Create the Bootable USB

    Click Start. Rufus will warn you that all data will be destroyed—confirm to continue. Wait for the progress bar to reach 100%. This may take several minutes.

    When done, you’ll see "READY" in Rufus. Safely eject your USB drive.

  6. 6

    Boot from the USB & Install Windows

    • Insert the USB into the target PC.
    • Restart the PC and enter the boot menu (usually F12, F2, DEL, or ESC at startup).
    • Select your USB drive as the boot device.
    • The Windows installer will start. Follow the on-screen instructions to install Windows.

    Tip: If you don’t see the USB, check your BIOS/UEFI settings and enable booting from USB.

  7. 7

    Complete Windows Setup

    After installation, your PC will restart. Remove the USB drive when prompted. Complete the Windows setup (region, keyboard, account, etc.).

    Congratulations! You’ve installed Windows using a USB and Rufus.

Frequently Asked Questions

Can I use this for Windows 10 or 8?

Absolutely! The process is identical for Windows 10, 8, or even 7—just use the correct ISO.

What if my PC doesn’t boot from USB?

Enter your BIOS/UEFI (usually by pressing F2, F12, DEL, or ESC at startup) and set the USB drive as the first boot device.

Is Rufus safe?

Yes, Rufus is a trusted, open-source tool used by millions worldwide.

What partition scheme should I use?

Use GPT for modern UEFI systems (most PCs from 2012 onward). Use MBR for older BIOS systems.

Can I use a Mac or Linux to create the USB?

This guide is for Windows. For Mac or Linux, use Balena Etcher or UNetbootin.
