Step-by-Step Guide

  1. 1

    Download Windows ISO

    Visit the official Microsoft page and download the Windows ISO file.

  2. 2

    Download & Open Rufus

    Go to rufus.ie, download Rufus, and launch it (no installation needed).

  3. 3

    Insert USB Drive

    Plug your USB drive into your PC. Backup any data on it, as it will be erased.

  4. 4

    Configure Rufus

    • Device: Select your USB drive.
    • Boot selection: Choose the Windows ISO.
    • Partition scheme: GPT (for UEFI) or MBR (for BIOS).
    • File system: NTFS or FAT32.
  5. 5

    Create Bootable USB

    Click Start in Rufus. Wait for the process to finish. Your USB is now ready!

  6. 6

    Boot & Install Windows

    Restart your PC, boot from the USB (change boot order in BIOS if needed), and follow the Windows installation wizard.
