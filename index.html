1. Prerequisites

Hardware Requirements

  • USB drive (8GB or larger)
  • Computer with internet connection
  • Computer capable of running Windows 10

Software Requirements

  • Windows 10 ISO file
  • Rufus application
  • Working computer to create the bootable USB

Warning: Creating a bootable USB will erase all data on the drive. Make sure to back up any important files.

2. Downloading Windows 10

1

Visit the Microsoft Download Page

Go to https://www.microsoft.com/software-download/windows10

2

Download the Windows 10 ISO

Scroll down and click "Download Now" under the "Create Windows 10 installation media" section.

Windows 10 ISO is approximately 5GB. The download may take 10-30 minutes depending on your internet speed.

3

Verify Your Download

Ensure your ISO file downloaded completely and isn't corrupted by checking the file size.

Expected file size: approximately 5GB

3. Rufus Setup Guide

1

Download Rufus

Download Rufus from the official website: https://rufus.ie

You can choose the portable version which doesn't require installation
2

Run Rufus

Double-click the Rufus executable file to run it. You may see a UAC (User Account Control) prompt - click Yes.

4. Creating Bootable USB

1

Insert Your USB Drive

Insert your USB drive (8GB or larger) into your computer.

Warning: All data on the USB drive will be erased during this process!

2

Configure Rufus

Set the following options in Rufus:

  • Device: Select your USB drive
  • Boot selection: Select "Disk or ISO image" and click SELECT
  • Choose the Windows 10 ISO file you downloaded
  • Partition scheme: GPT
  • Target system: UEFI (non CSM)
  • File system: NTFS
  • Cluster size: Default
3

Start the Process

Click START to begin creating the bootable USB.

You'll see warning messages about data being destroyed. Click OK to continue.

The process typically takes 10-15 minutes to complete.

5. BIOS Configuration

Accessing BIOS/UEFI

Common keys to access BIOS/UEFI during startup:

Manufacturer Key
Dell F2 or F12
HP F10 or Esc
Lenovo F1 or F2
ASUS F2 or Del
Acer F2 or Del
1

Change Boot Order

Navigate to the Boot settings in your BIOS/UEFI.

Change the boot order to prioritize USB boot over hard drive boot.

2

Save Changes

Save changes and exit BIOS/UEFI (usually F10).

Your computer will restart and boot from the USB drive.

6. Windows Installation Process

1

Windows Setup

The Windows Setup screen will appear. Select your language, time, and keyboard preferences, then click "Next".

2

Install Windows

Click "Install Now" to begin the installation process.

3

Product Key

Enter your product key or select "I don't have a product key" to continue.

4

Select Edition

Choose the Windows 10 edition you want to install.

5

Accept License Terms

Read and accept the license terms.

6

Installation Type

Choose "Custom: Install Windows only" for a clean installation.

7

Partition Selection

Select the partition where you want to install Windows.

Warning: Formatting a partition will delete all data on it!

8

Complete Installation

Follow the remaining prompts to complete the Windows 10 installation.

Your computer will restart several times during this process.

The installation process typically takes 15-40 minutes depending on your hardware.

7. Troubleshooting

If your USB drive is not detected during boot:

  • Check if the USB is properly inserted
  • Try a different USB port
  • Ensure USB boot is enabled in BIOS
  • Try creating the bootable USB again

This error often occurs when:

  • The disk is not properly formatted
  • The disk is using an unsupported partition style
  • The selected disk has MBR partition style but you're trying to install in UEFI mode

Solution: Format the disk using the Windows installer's built-in tools.

If the installation freezes or crashes:

  • Check for hardware compatibility issues
  • Disconnect unnecessary peripherals
  • Ensure your computer meets minimum system requirements
  • Try re-creating the USB installation media
  • Check if your RAM is functioning properly

8. Frequently Asked Questions

Do I need a product key to install Windows 10?

No, you can install Windows 10 without a product key. However, you will have limited personalization options and a watermark will appear on your desktop until you activate Windows.

Can I use Rufus on macOS?

No, Rufus is only available for Windows. For macOS, you can use alternative tools like BalenaEtcher or the Boot Camp Assistant.

How long does the entire process take?

The entire process typically takes 1-2 hours, including:

  • Downloading Windows 10 ISO: 10-30 minutes
  • Creating bootable USB: 10-15 minutes
  • Installing Windows: 15-40 minutes
  • Initial setup: 10-20 minutes

Times may vary depending on your internet speed and computer hardware.

Will this delete my files?

Creating a bootable USB will erase all data on the USB drive. When installing Windows, choosing "Custom: Install Windows only" and formatting/deleting partitions will erase all data on those partitions. Always back up important data before proceeding.
