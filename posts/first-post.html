Create a Windows 11 Bootable USB

April 2025 • Guide

  1. Download Windows 11 ISO:
    Microsoft's website
  2. Download Rufus:
    rufus.ie
  3. Insert USB (8GB+), open Rufus.
  4. Configure:
    • Select USB device
    • Choose ISO
    • Partition: GPT (UEFI)
    • File system: NTFS or FAT32
  5. Click Start. Wait for completion.
  6. Boot from USB, install Windows.
