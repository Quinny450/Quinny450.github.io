How to Create a Windows 11 Bootable USB

April 2025 • Guide

  1. Download the Windows ISO:
    Go to Microsoft's website and download the Windows 11 ISO.
  2. Download Rufus:
    Get Rufus from rufus.ie and open it.
  3. Insert your USB drive:
    Use at least an 8GB USB. Back up any important files first.
  4. Configure Rufus:
    • Select your USB device.
    • Choose the Windows ISO you downloaded.
    • Partition scheme: GPT (for most modern PCs).
    • File system: NTFS or FAT32.
  5. Create the bootable USB:
    Click Start and wait for Rufus to finish.
  6. Boot from USB and install Windows:
    Restart your PC, boot from the USB, and follow the Windows setup instructions.
